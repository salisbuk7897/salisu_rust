# Rust Tutorial Code
  I followed along rust lang crash course by [Traversy Media](https://www.youtube.com/watch?v=zF34dRivLOw) to learn and write this code. It contains the following files in the **src** folder:  
  - **main.rs** : The entry point of the project.  
  - **array.rs** : rust code for arrays.  
  - **cli.rs** : rust code for collecting and manipulating CLI arguments from User.  
  - **conditionals.rs** : rust code for conditionals.  
  - **enum.rs** : rust code for Enumeration.  
  - **funtions.rs** : rust code for function definition and usage.  
  - **loops.rs** : rust code for differnt types of loops.  
  - **pointer_refs.rs** : rust code for pointer reference.  
  - **string.rs** : rust code for strings.
  - **structs.rs** : rust code for structs.  
  - **tuples.rs** : rust code for tuples.  
  - **types.rs** : rust code for differnt data types.  
  - **vectors.rs** : rust code for vectors.  
  - **print.rs** : rust code for differnt types of print formatting.  
  - **vars.rs** : rust code for differnt types of print variables.
